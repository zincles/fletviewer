# fletviewer-native
FletviewerNative control for Flet

## Installation

Add dependency to `pyproject.toml` of your Flet app:

* **Git dependency**

Link to git repository:

```
dependencies = [
  "fletviewer-native @ git+https://github.com/MyGithubAccount/fletviewer-native",
  "flet>=0.85.3",
]
```

* **uv/pip dependency**

If the package is published on pypi.org:

```
dependencies = [
  "fletviewer-native",
  "flet>=0.85.3",
]
```

Build your app:
```
flet build macos -v
```
